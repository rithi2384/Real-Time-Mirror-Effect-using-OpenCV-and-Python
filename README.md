Virtual Mirror using OpenCV

This project uses OpenCV and Python to create a real-time virtual mirror using a webcam. The program captures video from the camera and flips the frame horizontally to simulate a mirror reflection.

✨ Features

Real-time webcam mirror display

Horizontal frame flipping to simulate mirror effect

Simple computer vision implementation

🛠 Technologies Used

Python

OpenCV

📌 Applications

Computer vision learning

Webcam-based applications

Image processing demonstrations
